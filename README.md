# Busca de Endereço com ViaCEP

Aplicação ASP.NET MVC que utiliza:

- Model Binding
- jQuery + AJAX
- API ViaCEP
- Controller Server Side

O usuário digita o CEP, os dados são preenchidos automaticamente via AJAX e enviados ao servidor. O controller processa os dados e redireciona para uma tela de resultado somente leitura.
